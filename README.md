##  Fase 1 - Definición del Proyecto

###  Objetivo
Crear una agenda digital que ayude a llevar el control de:
- Vacunas
- Citas veterinarias
- Alimentación
- Recordatorios importantes para mascotas

![](/prueba.png)

---

##  Descripción
La Agenda Inteligente de Mascotas es una herramienta diseñada para dueños de mascotas que desean organizar mejor el cuidado de sus animales.

Permitirá registrar información y recibir recordatorios para mejorar la salud y bienestar de las mascotas.

---

##  Funcionalidades (Fase 1)
- Registro de mascotas
- Guardar nombre, edad y tipo de mascota
- Registrar fechas importantes
- Organización básica de información

---

##  Tecnologías (planeadas)
- C#
- HTML



---

## Estructura inicial
- Interfaz simple
- Datos organizados tipo tabla (similar a Excel)
